HTML

  • The head element contains information about the webpage
  • The body element represents the visable content shown to the user.
  • ul is unordered list. ol is organized list
  • li is list item and must be done for each point

CSS

  • inline CSS:apply to a specific HTML tag
  • internal style sheet: CSS rules for the webpage in the head section
  • external CSS style sheet: keeps CSS rules in a different file
  • linking style sheet to HTML file: rel="stylesheet" href="./assets/style.css"
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • css rule: selector {declaration}. You can have multiple declarations to one selector but seperate using ;. (div {color: purple; text-align: right}
